
# README.md – My Portfolio
This is the source code for my personal portfolio website, built with modern tools and deployed on Vercel.

Tech Stack

Vite – Ultra-fast frontend tooling
React – UI library
TypeScript – Type safety
Tailwind CSS – Utility-first styling
shadcn/ui – Beautiful, accessible components built on Radix UI + Tailwind

Local Development
Prerequisites

Node.js ≥ 18 (recommended: use nvm)
npm or pnpm or yarn

Setup
Bash# 1. Clone the repository
git clone https://github.com/your-username/your-portfolio.git
cd your-portfolio

# 2. Install dependencies
npm install
# or if you prefer pnpm/yarn
# pnpm install
# yarn

# 3. Start the development server
npm run dev
Open http://localhost:5173 to see your site with hot-reload.
Other useful scripts
Bash# Build for production
npm run build

# Preview the production build locally
npm run preview
